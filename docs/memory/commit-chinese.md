## Git 提交规范

- commit message 使用中文描述，不用英文
- 文档作者名使用 **Fang**（不用方瀚然）

**Why:** 用户偏好中文沟通，commit 描述也应保持一致；作者名统一为 Fang。
**How to apply:** 
1. 所有 `git commit -m` 的内容用中文撰写
2. 文档中的作者名写 Fang，不写方瀚然
