## 3. 设计意图文档

README 中补充以下小节：

- **为什么用 Protocol 依赖注入** — 端侧/云端/边缘设备可插拔切换检测器和追踪器
- **为什么预留 RAG 接口** — 后期接入知识库，LLM 分析时参考历史案例和 SOP
- **为什么三层队列解耦** — 防止 GPU 慢导致摄像头丢帧，防止单个摄像头断线拖垮整体
- **端侧接口的预留思路** — 当前 Web API 服务于云端，端侧可复用 core 层

**目的**：让读者理解架构背后的思考，而不只是看到代码。
